What is JIT?

Just-In-Time (JIT) compilation is a technique that improves the performance of apps by compiling code during execution rather than in advance. This is especially important for:

  • Emulators (like Delta, PPSSPP, UTM)
  • JavaScript engines
  • Virtual machines
  • Programming language interpreters

iOS normally restricts JIT compilation for security reasons, but our JIT Enabler Shortcut allows you to enable it for your apps.
