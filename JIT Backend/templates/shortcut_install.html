Installation Instructions:

For Signed Shortcut:

  1. Click the "Download Signed Shortcut" button or scan the QR code with your iOS device camera
  2. When prompted, tap "Add Shortcut"
  3. If asked, confirm that you trust the shortcut
  4. The shortcut will be added to your Shortcuts app

For Jellycuts Version:

  1. Click the "Download Jellycuts Version" button
  2. Follow the instructions in the Jellycuts README to install the shortcut
